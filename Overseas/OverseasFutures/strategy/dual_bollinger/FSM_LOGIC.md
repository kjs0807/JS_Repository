# Dual Bollinger Band FSM 전략 로직 상세 해설

## 용어 정리

| 용어 | 의미 |
|------|------|
| **inner band** | 볼린저 밴드 (MA ± sigma_inner × 표준편차). 횡보/추세 경계선 역할 |
| **outer band** | 확장 볼린저 밴드 (MA ± sigma_outer × 표준편차). 과매수/과매도 영역 |
| **ATR** | Average True Range. 최근 14봉의 평균 변동폭 (변동성 측정) |
| **RSI** | Relative Strength Index (0~100). 70 이상 과매수, 20 이하 과매도 |
| **bandwidth** | 밴드 폭 = (upper - lower) / MA × 100%. 변동성이 클수록 큼 |
| **trailing stop** | 수익이 일정 수준 이상 나면 활성화되어, 최고 수익 지점에서 일정 거리를 유지하며 따라오는 손절선 |
| **base_qty** | 1차 진입 수량 (3Y: 100계약, 10Y: 30계약) |
| **scale_qty** | 2차 진입 수량 (3Y: 100계약, 10Y: 30계약) |
| **크로스오버** | 이전 봉의 종가와 현재 봉의 종가를 비교해서 특정 레벨을 "넘었는지" 감지 |

---

## 현재 파라미터 (240m, KTBF)

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| MA period | 5 | 최근 5봉의 이동평균 |
| sigma_inner | 1.0 | inner band = MA ± 1.0 × 표준편차 |
| sigma_outer | 2.0 | outer band = MA ± 2.0 × 표준편차 |
| breakout_pct | 0.0% | 돌파 최소 마진 (0이면 그냥 넘기만 하면 됨) |
| atr_stop_multiplier | 1.5 | 손절 거리 = ATR × 1.5 |
| trailing_activation_atr | 1.0 | 트레일링 활성화 = 수익 > ATR × 1.0 |
| trailing_distance_atr | 1.5 | 트레일링 거리 = ATR × 1.5 |
| rsi_overbought | 70 | RSI 과매수 기준 |
| rsi_oversold | 20 | RSI 과매도 기준 |
| max_bandwidth_pct | 8.0% | 밴드폭 > 8%면 진입 금지 |

---

## 7개 상태 (State)

```
FLAT ─────────── 포지션 없음
LONG_1ST ─────── 롱 1차 진입 (base_qty 보유)
LONG_2ND ─────── 롱 2차 진입 (base_qty + scale_qty 보유)
LONG_PARTIAL ─── 롱 절반 청산 후 (scale_qty만 보유)
SHORT_1ST ────── 숏 1차 진입 (base_qty 보유)
SHORT_2ND ────── 숏 2차 진입 (base_qty + scale_qty 보유)
SHORT_PARTIAL ── 숏 절반 청산 후 (scale_qty만 보유)
```

---

## SHORT 흐름 (상세)

### 1단계: FLAT → SHORT_1ST (1차 진입)

**언제?**
- 이전 봉 종가가 inner band 하단 **위**에 있었고
- 현재 봉 종가가 inner band 하단 **아래**로 내려감
- 변동성 필터 통과 (밴드폭 < 8%)

**쉽게 말하면**: 가격이 횡보 구간(inner band 안)에서 아래로 벗어남 = "하락 추세 시작 신호"

**진입 가격**: 현재 봉의 종가

**손절가 설정**:
- 후보 A: 종가 + ATR × 1.5 (변동성 기반)
- 후보 B: inner band 상단 (밴드 기반)
- **둘 중 더 낮은 값** 사용 (= 더 타이트한 손절)
- 예시: 종가 105.28, ATR=0.15일 때
  - 후보 A: 105.28 + 0.15 × 1.5 = 105.505
  - 후보 B: inner_upper = 105.45
  - 손절가 = **105.45** (더 낮으니까)

**수량**: base_qty (3Y: 100계약)

---

### 2단계: SHORT_1ST 상태에서 매 봉마다 체크

아래 순서대로 체크하고 **먼저 걸리면 그 아래는 안 봄**:

#### 체크 1: 트레일링 스탑 (우선순위 최상)
- 트레일링이 활성화되어 있고, 가격이 트레일링 레벨 위로 올라가면 → **전량 청산**
- (자세한 트레일링 메커니즘은 아래 별도 섹션)

#### 체크 2: Outer Band + RSI 조기 익절
- 종가가 outer band 하단 **아래**까지 떨어지고 + RSI가 20 이하 (과매도)
- → **전량 청산** (과매도 영역이니 더 떨어지기 어려움)
- 실제로는 거의 안 걸림 (RSI 20 이하가 매우 드뭄)

#### 체크 3: 손절
- 종가가 손절가 **위**로 올라감
- → **전량 청산** (추세 실패, 손실 확정)

#### 체크 4: 2차 진입 조건
- 종가가 inner band 하단 **위**로 복귀 (= 풀백/되돌림)
- → **SHORT_2ND**로 전환, scale_qty만큼 추가 매도
- 평균 단가 재계산
- **손절가는 변경하지 않음** (1차 진입 때 설정한 그대로)

**쉽게 말하면**:
- 1차 진입 후 가격이 내려갔다가 다시 올라오면 = "눌림목"
- 이때 추가로 물량을 실어서 평단을 맞춤
- 만약 올라오다가 손절가를 넘기면 체크 3에서 먼저 손절됨

---

### 3단계: SHORT_2ND 상태에서 매 봉마다 체크

#### 체크 1: 트레일링 스탑 → 전량 청산
#### 체크 2: Outer + RSI → 전량 청산
#### 체크 3: 손절 → 전량 청산

#### 체크 4: Partial Exit (절반 청산)
- 종가가 inner band 하단 **아래**로 다시 내려감
- → **base_qty만큼 청산** (절반 이익 실현)
- → **SHORT_PARTIAL** 상태로 전환

**쉽게 말하면**:
- 2차 진입(눌림목) 후 가격이 다시 inner band 아래로 내려감
- = "눌림목이었고, 추세가 다시 살아남" 확인
- 이 시점에 절반을 먹음 (리스크 관리)
- 나머지 절반은 추세를 더 끌고 감

**논란 포인트**:
- 추세가 살아있는데 왜 절반을 줄이느냐?
- → 답: 240m 봉 기준 백테스트에서 이 로직이 수익률/샤프 최고
- → 추세 약화 시 partial exit으로 변경하면 성과 급락 (검증 완료)

---

### 4단계: SHORT_PARTIAL 상태에서 매 봉마다 체크

잔여 물량(scale_qty)을 들고 있는 상태.

#### 체크 1: 트레일링 스탑 → 잔여 전량 청산

#### 체크 2: Full Exit
- 종가가 inner band **상단** 위로 올라감 **또는** 손절가 위로 올라감
- → **잔여 전량 청산**
- 의미: 추세가 완전히 반전됨

#### 체크 3: Band Exit (크로스오버)
- **이전 봉** 종가 < inner band 하단 (아직 추세 안)
- **현재 봉** 종가 >= inner band 하단 (밴드 안으로 복귀)
- → **잔여 전량 청산**
- 의미: inner band를 아래에서 위로 다시 넘어옴 = 하락 추세 종료 신호

---

## LONG 흐름 (상세)

### 1단계: FLAT → LONG_1ST (1차 진입)

**언제?**
- 이전 봉 종가가 inner band 상단 **아래**에 있었고
- 현재 봉 종가가 inner band 상단 **위**로 올라감

**손절가**: `max(종가 - ATR × 1.5, inner_lower)` — 둘 중 **더 높은 값** (= 더 타이트한 손절)

**수량**: base_qty

---

### 2단계: LONG_1ST 체크

| 순서 | 조건 | 결과 |
|------|------|------|
| 1 | 트레일링 스탑: 종가 <= 트레일링 레벨 | 전량 청산 → FLAT |
| 2 | 종가 >= outer_upper AND RSI >= 70 | 전량 청산 → FLAT (과매수) |
| 3 | 종가 < 손절가 | 전량 청산 → FLAT |
| 4 | 종가 <= inner_upper (풀백) | 2차 진입 → LONG_2ND |

---

### 3단계: LONG_2ND 체크

| 순서 | 조건 | 결과 |
|------|------|------|
| 1 | 트레일링 스탑 | 전량 청산 → FLAT |
| 2 | outer_upper + RSI 과매수 | 전량 청산 → FLAT |
| 3 | 종가 < 손절가 | 전량 청산 → FLAT |
| 4 | 종가 > inner_upper (추세 재확인) | 절반 청산 → LONG_PARTIAL |

---

### 4단계: LONG_PARTIAL 체크

| 순서 | 조건 | 결과 |
|------|------|------|
| 1 | 트레일링 스탑 | 잔여 전량 청산 → FLAT |
| 2 | 종가 < inner_lower 또는 종가 < 손절가 | 잔여 전량 청산 → FLAT (full_exit) |
| 3 | 이전봉 > inner_upper → 현재봉 <= inner_upper (크로스) | 잔여 전량 청산 → FLAT (band_exit) |

---

## LONG vs SHORT 대칭 확인

| 항목 | LONG | SHORT | 대칭? |
|------|------|-------|-------|
| 진입 조건 | close > inner_upper (상향 돌파) | close < inner_lower (하향 돌파) | O |
| 크로스오버 | prev <= inner_upper → now > | prev >= inner_lower → now < | O |
| 손절가 | max(close - ATR×1.5, inner_lower) | min(close + ATR×1.5, inner_upper) | O |
| 2차 진입 | close <= inner_upper (풀백) | close >= inner_lower (풀백) | O |
| 2차 손절 재설정 | 안 함 | 안 함 | O |
| Partial exit | close > inner_upper (추세 재확인) | close < inner_lower (추세 재확인) | O |
| Full exit (PARTIAL) | close < inner_lower | close > inner_upper | O |
| Band exit (PARTIAL) | prev > inner_upper → now <= | prev < inner_lower → now >= | O |
| 트레일링 방향 | 최고가 추적, 하락 시 청산 | 최저가 추적, 상승 시 청산 | O |
| Outer+RSI | close >= outer_upper + RSI >= 70 | close <= outer_lower + RSI <= 20 | O |

**결론: LONG과 SHORT는 완전히 대칭**

## 트레일링 스탑 메커니즘 (상세)

### 1. 매 봉마다 업데이트 (포지션 보유 중)

**SHORT 예시** (ATR = 0.15):

```
활성화 거리 = ATR × 1.0 = 0.15
트레일링 거리 = ATR × 1.5 = 0.225
```

**최저가 추적**:
- 봉이 올 때마다 `최저가 = min(기존 최저가, 현재 종가)`
- 가격이 내려갈수록 최저가가 갱신됨

**활성화 조건**:
- `평균진입가 - 최저가 >= 0.15` (수익이 ATR 1배 이상)
- 예시: 진입 105.29, 최저가 105.10 → 수익 0.19 >= 0.15 → **활성화**

**트레일링 레벨 계산**:
- `트레일링 = 최저가 + ATR × 1.5 = 105.10 + 0.225 = 105.325`
- 최저가가 갱신되면 트레일링도 **내려감** (유리한 방향으로만 이동)
- 예시:
  - 최저가 105.10 → 트레일링 105.325
  - 최저가 105.05로 갱신 → 트레일링 105.275 (내려감)
  - 최저가 105.08 (갱신 안 됨) → 트레일링 105.275 (유지)

### 2. 발동 조건

- SHORT: `종가 >= 트레일링 레벨` → 전량 청산
- LONG: `종가 <= 트레일링 레벨` → 전량 청산

**쉽게 말하면**:
- 수익이 충분히 나면 "안전장치"가 켜짐
- 가격이 내 유리한 방향으로 갈수록 안전장치도 따라옴
- 가격이 반전해서 안전장치에 닿으면 자동 청산
- = "이만큼 벌었으면 여기까지는 지키자"

### 3. 트레일링 vs 손절 관계

- 손절가는 **고정** (진입 시 설정, 변경 없음)
- 트레일링은 **수익이 나야 활성화**, 이후 유리한 방향으로만 이동
- 활성화 전: 손절가만 작동
- 활성화 후: 트레일링이 손절가보다 유리한 위치에 있으므로 트레일링이 먼저 발동

## 청산 종류 요약

| 청산 유형 | 발동 상태 | 의미 | 성격 |
|-----------|----------|------|------|
| **trailing_stop_exit** | 1ST, 2ND, PARTIAL | 충분한 수익 후 반전 시 자동 청산 | 수익 보호 |
| **outer_rsi_exit** | 1ST, 2ND | 극단적 과매수/과매도 도달 | 수익 확정 |
| **stop_loss** | 1ST, 2ND, (PARTIAL 내 full_exit에 포함) | 진입 실패, 손실 확정 | 손실 제한 |
| **partial_exit** | 2ND → PARTIAL | 추세 재확인 후 절반 이익 실현 | 리스크 축소 |
| **full_exit** | PARTIAL | 추세 완전 반전 (inner band 반대편 도달) | 최종 청산 |
| **band_exit** | PARTIAL | inner band 복귀 크로스오버 | 최종 청산 |

## 실제 거래 시나리오 예시

### 시나리오 A: 이상적인 숏 트레이드

```
봉1: 가격 105.30 (inner_lower=105.32 위에 있음)
봉2: 가격 105.25 → inner_lower(105.32) 하향 돌파!
     → SHORT_1ST 진입 @ 105.25, 100계약
     → 손절가 = min(105.25 + 0.225, 105.50) = 105.475

봉3: 가격 105.15 (계속 하락, inner_lower 아래 유지)
     → 아무 조건도 안 걸림, 보유 유지

봉4: 가격 105.30 → inner_lower(105.31) 위로 복귀!
     → 2차 진입 @ 105.30, 100계약 추가
     → 총 200계약, 평단 105.275

봉5: 가격 105.10 → inner_lower(105.28) 아래로 다시 하락!
     → Partial exit: 100계약 청산 @ 105.10
     → 실현 수익: (105.275 - 105.10) × 100 × 100만 = +17,500,000원
     → 잔여 100계약 → SHORT_PARTIAL

봉6~10: 가격 계속 하락, 104.90까지
     → 트레일링 활성화, 트레일링 레벨 = 104.90 + 0.225 = 105.125

봉11: 가격 105.15 → 트레일링(105.125) 위로!
     → 잔여 100계약 청산 @ 105.15
     → 추가 실현 수익: (105.275 - 105.15) × 100 × 100만 = +12,500,000원
     → FLAT으로 복귀

총 수익: +30,000,000원
```

### 시나리오 B: 손절 트레이드

```
봉1: 가격 105.30 (inner_lower 위)
봉2: 가격 105.25 → inner_lower 하향 돌파!
     → SHORT_1ST 진입 @ 105.25, 손절가 105.475

봉3: 가격 105.50 → 손절가(105.475) 위로!
     → 손절: 100계약 청산 @ 105.50
     → 손실: (105.25 - 105.50) × 100 × 100만 = -25,000,000원
     → FLAT
```

### 시나리오 C: 2차 진입 없이 트레일링으로 끝남

```
봉2: SHORT_1ST 진입 @ 105.25
봉3: 105.10 (하락 지속, inner_lower 아래 유지)
봉4: 104.95 (하락 지속)
봉5: 104.80 → 트레일링 활성화 (수익 0.45 > ATR 0.15)
     → 트레일링 레벨 = 104.80 + 0.225 = 105.025
봉6: 105.05 → 트레일링(105.025) 돌파!
     → 전량 청산 @ 105.05
     → 수익: (105.25 - 105.05) × 100 × 100만 = +20,000,000원
     → 2차 진입 없이 1차만으로 끝남
```
